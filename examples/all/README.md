# Explanation

In this folder,

1. One third library (`third/`)
2. One Rosetta registration (`registration.h`)
3. Multiple bindings / tests:
   1. **Python** (`py/binding.cxx`)
   2. **JavaScript** (`js/binding.cxx`)
   3. **Emscipten** (`em/binding.cxx`)
   4. **Cpp**: pure C++, no introspection (`cpp/`)
   5. **Rosetta**: using only rosetta introspection (`rosetta/`)
